Resume Materi KMFlutter – Installation – Command Line Interface

1. Fluuter CLI merupakan perintah yang dijalankan pada terminal untuk berinteraksi dengan flutter sdk
2. Beberapa CLI yang dipelajari yaitu :
    - Flutter Doctor : digunakan untuk melihat spesifikasi apa saja yang dibutuhkan pada flutter. dengan perinttah ini kita dapat mengetahui apa saja yang harus di install untuk melengkapi instalasi flutter 
    - Flutter create : digunakan untuk membuat project baru
    - Flutter Run : untuk menjalankan prpject flutter
    - Flutter emulator : digunakan untuk menampilkan dan memilih emulator yang tersedia 
    - Flutter Channel : untuk menampilkan daftar channel yang tersedia dan channel yang sedang dipakai
    - Flutter pub 
        ---> Flutter pub add <package name> : digunakan untuk menambahkan packages ke dependencies
        ---> Flutter pub get : digunakan untuk mendownload semua packages/dependencies 
    - Flutter build : untuk membangun sebuah aplikasi agar dapat di publish. terdapat 2 jenis yaitu :
        ---> Flutter build apk : untuk android
        ---> Flutter build ios : untuk ios
    - Flutter Clean : digunakan untuk menghapus file-file yang dihasilkan saat running project. Setelah melakukan perintah ini kita dapat menjalankan flutter pub get kembali.
